# FieldOpsTracker

FieldOpsTracker is a full-stack web application built with **React (TypeScript)** and **.NET 8 Web API**, designed as a modern, cloud-ready baseline for operational tracking systems.

This project demonstrates a clean frontend–backend separation, HTTP-based integration, and development best practices suitable for production environments.

---

## Tech Stack

### Frontend
- React
- TypeScript
- Vite

### Backend
- .NET 8
- ASP.NET Core Web API
- Minimal APIs

### Tooling / Practices
- Git (mono-repo)
- RESTful API design
- CORS configuration for local development
